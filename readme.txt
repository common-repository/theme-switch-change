== Description ==
Are you addicted to Wordpress Theme? If that is the case this plugin could be your great friend.

This plugin allows you to change Theme to the administrator or user with permission to change Theme `(𝑠𝑤𝑖𝑡𝑐ℎ_𝑡ℎ𝑒𝑚𝑒𝑠)` the possibility to see how a post or any page on different Themes installed quickly.
This will only affect the administrator temporarily, visitors or users will see the Theme activated.

## ★ What would this plugin do for me?
It is very useful, will list some of them:

- You can quickly test and define a Theme for your page without affecting visitors, then you can activate the one you decided.
- If you are a developer of a plugin you can verify if your development is compatible with different Themes, this is important to know the degree of general compatibility.
- For other various purposes.

## ★ Features
- Show Parent and Child Themes
- Preview screenshot when you hover over the theme name
- Show author information and theme version together to preview
- Without complications, it does not register anything in your database.
- Lightweight, fast and flexible.
- Plugin cleans up after itself upon uninstall.
- Simple and focused.
- Unique and minimalist design.
- Change the theme quickly

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Theme Switch Change'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. You can see it in action in the Menu Bar

= Uploading in WordPress Dashboard =

1. Download `theme-switch-change.zip` from this page
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `theme-switch-change.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard
